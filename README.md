# 🚚 Sansam Fleet - Internal Management App

Aplikasi Progressive Web App (PWA) berbasis web yang dirancang khusus untuk manajemen internal logistik, pelacakan ritase armada dump truck, absensi driver, dan kontrol arus kas operasional pada **CV SANSAM GROUP**.

## ✨ Fitur Utama
* **Dashboard Real-Time:** Memantau total pendapatan, sisa kas di driver, panjar aktif, dan pelunasan tempo secara langsung.
* **Input Ritase & Kas Cepat:** Form input data perjalanan armada dilengkapi fitur *quick-fill* muatan dan rute.
* **Manajemen Kas & Panjar Driver:** Sistem pencatatan solar, sparepart, transfer kas antar driver, serta metode sinkronisasi potong kas harian.
* **Visualisasi Data:** Grafik distribusi ritase driver interaktif menggunakan ChartJS.
* **Mesin Absensi Driver:** Pencatatan kehadiran dan kondisi operasional driver harian secara real-time.
* **Premium Export:** Mengunduh rekapan log data langsung ke format Excel (CSV) atau dokumen PDF resmi lengkap dengan Kop Surat CV SANSAM GROUP.
* **PWA Support:** Dapat diinstal langsung di layar utama perangkat Android maupun iOS untuk pengalaman seperti aplikasi native.

## 🛠️ Teknologi yang Digunakan
* Frontend: HTML5, Tailwind CSS, FontAwesome Premium Icons.
* Library JS: Chart.js (Grafik), jsPDF & jsPDF AutoTable (Ekspor Dokumen).
* Fitur Mobile: Web App Manifest Dinamis & Service Worker (PWA) untuk mendukung mode instalasi HP.
